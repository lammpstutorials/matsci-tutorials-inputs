## Input files for Materials Science LAMMPS tutorials

LAMMPS and Python input files corresponding to
[LAMMPS Materials Science Tutorials](https://lammpstutorials/matsci-tutorials-article/).

The files include:

- LAMMPS data files, input files, and parameter files,
- the detailed solutions to the exercises,
- Python scripts for generating structures and analyzing data.

### Main repository

Find [here](https://github.com/lammpstutorials/lammpstutorials.github.io)
the main repository for all LAMMPS tutorials in this collection after
they are published.
